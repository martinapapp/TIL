# Computer Hardware Components

## Computer Components For Dummies
*(link)[https://www.youtube.com/watch?v=cZs6kh0WFRY]*

**Computer Parts List:**

- motherboard
- CPU
- RAM
- GPU
- Hard Drive/SSD


**CPU** is the Central Processing Unit. Brain is the computer. Dictates how fast the computer is. Responsible arithmetic, math, heavy lifting the computer. Utalize all the other parts. 2 brand: AMD and Intel. AMD outperforming intel the value for CPUs. Looks for the specs when choosing: number of cores (min. i4 core), core speed (e.g. 4GHz, speed of a single core, depends on base/boost clock), neumber of threads (double-s the number of cores), generation.

**RAM** is a Random Access Memory. Temporarly storage location. When CPU is turned off everything is going to be disapear. It is good for quickly access and change. DDR4 !!! no DDR3. 8-16GB.

**Motherboard** is the bridge, connection between components. Components communicates through the bus on the motherboard. It will also deliver power to the components, unless the component has its own separate power connection. Motherpard CPU compability is important. It has input and output devices (usb jack, hdmi, vga port).

**GPU** is the Graphics Processing Unit. For gaming and professional usecases use:

**Dedicates GPU**. Otherwise built-in in the CPU is fine. Not all CPU has integrated graphics (like most AMD, (intel mostly has)). 2 prefered companies: AMD and NVIDIA. First check of the specs is the generation of the GPU - go for most recent. Look at the VRAM (=video random access memory) which is a special dedicated memory, like RAM, but only for the graphics card. VRAM acts like a video buffer - everything you see is being stored in the VRAM first and than passed to the display output where you will acutally see it on the screen. Double the RAM of the VRAM. 4GB is fine for light video editing, for more complex use 6-8-12-24GB. It needs a lot of power supply (300watts).

**Hard drive** is a physical, spinning, magnetic disk which has read and write head taht's able to read magnetic charges of the disk. (if magnet is near to the hard drive, it will erase all of the data, beacuse it magnetically charged) and it is lot slower than a sloid slate drive, beacuse it phisically need to read magnetic bits rigth from the hard drive. Advantage is cheaper and have a huge storage capacity. It is good for data drive.

**SSD** is a Solid State Drive. Fast storage medium, but much more expensive and has lower capacity. If a computer has 1 drive, it has to be SSD for anything you load and use frequently! On hard drive the load time is 10 times longer.
