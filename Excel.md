### Terminology


workbook: highest-level component, represented as a .xlsx file, contains data, calculations, functions, underlying element. Consist one or more worksheets, represented on a tab. Tabs can be renamed, change order.

worksheet: made up of cells, which contains data (text, numbers, formulas, calculation results). organized in columns and rows => cell reference (first column, second row).

cell range: a collection of several cells selected together. Can be made by mouse selecting or SHIFT+arrow keys(= arrays, most common for ref in calcs and formulas). notation examples: *D9:D19*, *D9:H9*,
*D9:H19*

### Navigation

shortcuts:
# Keyboard Shortcuts

| Task | Shortcut |
|------|----------|
| Shut the current workbook | Ctrl+W |
| Launch a workbook | Ctrl+O |
| Write changes to disk | Ctrl+S |
| Duplicate selection | Ctrl+C |
| Remove and copy selection | Ctrl+X |
| Insert from clipboard | Ctrl+V |
| Reverse last action | Ctrl+Z |
| Clear selected cells | Delete |
| Toggle bold formatting | Ctrl+B |
| Trigger right-click menu | Shift+F10 |
| Show or hide the ribbon | Ctrl+F1 |
| Go up one cell | ↑ |
| Go down one cell | ↓ |
| Go left one cell | ← |
| Go right one cell | → |
| Jump to the boundary of the current data block | Ctrl+Arrow key (e.g. Ctrl+↓) |
| Jump to the furthest used cell on the sheet | Ctrl+End |
| Jump to the top of the sheet | Ctrl+Home |
| Expand selection to the last active cell | Ctrl+Shift+End |
| Return to the top-left visible cell (requires Scroll Lock) | Home+Scroll Lock |
| Scroll down one full screen | Page Down |
| Scroll up one full screen | Page Up |
| Scroll right one full screen | Alt+Page Down |
| Scroll left one full screen | Alt+Page Up |
| Switch to the next sheet | Ctrl+Page Down |
| Switch to the previous sheet | Ctrl+Page Up |
| Edit the active cell from the end | F2 |
| Stamp the current time | Ctrl+Shift+: |
| Stamp the current date | Ctrl+; |
